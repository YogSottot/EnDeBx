# Upgrade дистрибутивов

Этот раздел собирает сценарии обновления мажорных версий ОС поверх уже установленного окружения EnDeBx.

## Как пользоваться разделом

Сценарии upgrade в этом проекте лучше воспринимать как практические runbook-инструкции:

- сначала обновить меню и проверить совместимость окружения;
- затем выполнить подготовительные шаги;
- только после этого переходить к upgrade самой ОС.

## Общие рекомендации

Перед любым major-upgrade полезно:

- сделать полноценный backup;
- проверить сценарий на тестовой копии;
- использовать `screen`, `tmux` или `byobu`;
- заранее проверить поддержку вашей ветки MySQL/PostgreSQL на целевом релизе ОС.

Сейчас в репозитории оформлен сценарий для [Debian 12 -> 13](debian-12-to-13.md).
