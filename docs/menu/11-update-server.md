# 11. `Update server`

Это самый простой системный сценарий в меню.

## Что делает пункт

После подтверждения запускаются:

```bash
apt update -y
apt upgrade -y
```

## Что этот пункт не делает

Он не выполняет:

- `apt full-upgrade`;
- cleanup старых пакетов;
- reboot после обновления.

Если после апдейта нужен reboot, это уже отдельное действие из главного меню.

## Когда использовать

- для регулярного обновления сервера;
- перед установкой новых расширений;
- перед upgrade дистрибутива, если нужна чистая актуальная база пакетов.
