# Системные репозитории и Snapd

Здесь собраны небольшие, но потенциально чувствительные системные переключатели.

## `Install/Delete Snapd`

Пункт ставит или удаляет `snapd`. Используйте только, если вы понимаете, что делаете. Все данные установленных из snap приложений будут удалены

## `Install/Delete Deadsnakes PPA`

Подключение [Deadsnakes PPA](https://launchpad.net/~deadsnakes/+archive/ubuntu/ppa). Пункт отображается только на Ubuntu. Он подключает или удаляет PPA с более свежими версиями Python.

!!! warning "Не для Debian"
    Меню само показывает этот пункт только на Ubuntu. Использовать Ubuntu PPA в Debian не рекомендуется.

## `Install/Delete Debian repo on Astra Linux`

Пункт отображается только на Astra Linux и временно включает стандартные Debian-репозитории для Astra-сценариев.

Такой переключатель лучше использовать только если вы понимаете, зачем именно нужен внешний Debian-репозиторий на Astra.
