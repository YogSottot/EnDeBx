# `R` - перезагрузка сервера

В главном меню остается прямое системное действие:

- `R` - перезагрузка сервера.

## `R` - Restart the server

После подтверждения меню выполняет:

```bash
reboot
```

## Когда быть особенно осторожным

Перед запуском этих пунктов стоит убедиться, что:

- у вас есть доступ к консоли или out-of-band управлению;
- не выполняются миграции БД, backup или deploy;
- вы понимаете, кто еще работает на этом сервере прямо сейчас.
